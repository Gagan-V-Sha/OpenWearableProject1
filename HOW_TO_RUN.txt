  pip install -r requirements.txt

  python -m uvicorn app.main:app --host 127.0.0.1 --port 8000

  http://127.0.0.1:8000

  data/raw/          Original LifeSnaps CSV files (unprocessed export)
  data/processed/    Cleaned daily table used by the app (see DATA_README.txt)
